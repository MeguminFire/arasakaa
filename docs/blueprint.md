# **App Name**: Troubleshoot Titans

## Core Features:

- Troubleshooting Game: Presents the user with a simulated computer issue and has them go through troubleshooting steps to solve the issue. Generative AI serves as a tool, checking user answers and providing appropriate feedback. A leader board ranks each user on time and successful completion.
- Troubleshooting Quiz: Presents the user with a multiple choice quiz on troubleshooting topics, with leader board ranking user score and completion time.
- Progress Tracking: Tracks user progress through the games and quizzes.
- Progress Bar: Displays a progress bar indicating completion for each game.
- Title Rewards: Award the user a title when they complete a game or quiz.
- Performance Stats: Capture game performance (time, score, completion, etc) and make it available to the user.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent technology and problem-solving.
- Background color: Light gray (#F0F0F0) for a clean and professional look.
- Accent color: A bright orange (#FF9933) to highlight important elements and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use icons related to computers, networking, and troubleshooting.
- Use a clean and organized layout with clear sections for games, quizzes, and stats.
- Subtle animations for user feedback and progress indication.
